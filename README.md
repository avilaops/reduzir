# Reduzir PDF - JLA Importadora

Sistema para redução de tamanho de PDFs e envio de emails com catálogos.

## Funcionalidades

- **Compressão de PDF**: Reduz o tamanho de arquivos PDF convertendo páginas em imagens JPEG otimizadas
- **Email Marketing HTML**: Template profissional para envio de catálogos por email
- **Envio Automatizado**: Script para envio de emails via SMTP com anexos (catálogo completo 121.9 MB)
- **Download Rápido**: [📥 Versão Otimizada](https://github.com/avilaops/reduzir/raw/main/Catalogo-JLA-reduzido.pdf) (7.20 MB)

## Arquivos Principais

- `compress_pdf_aggressive.py`: Script de compressão de PDF
- `email-catalogo.html`: Template HTML do email
- `enviar_email.py`: Script de envio de emails
- `image.png`: Logo da empresa

## Uso

### Comprimir PDF

```bash
python compress_pdf_aggressive.py
```

### Enviar Email

```bash
python enviar_email.py
```

## Requisitos

- Python 3.10+
- PyMuPDF (fitz)
- Pillow
